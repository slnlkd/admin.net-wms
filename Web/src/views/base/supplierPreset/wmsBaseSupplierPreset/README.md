# 供应商信息申请

## 功能概述
供应商信息申请功能允许业务员提交新的供应商信息申请，管理员可以审核这些申请。

## 功能模块

### 1. 业务员端 - 申请供应商信息
- **路径**: `/views/base/supplierPreset/wmsBaseSupplierPreset/index.vue`
- **功能**:
  - 新增供应商信息申请
  - 编辑已提交的待审核和驳回状态的申请
  - 删除已提交的待审核和驳回状态的申请
  - 查看审核状态和审核备注
  - 导入和导出功能

### 2. 管理员端 - 供应商信息管理
- **路径**: `/views/base/baseSupplier/wmsBaseSupplier/index.vue`
- **功能**:
  - 查看所有供应商信息（包括已审核通过的）
  - 审核供应商信息申请
  - 审核按钮显示待审核数量
  - 审核弹窗支持批量审核

### 3. 审核弹窗
- **路径**: `/views/base/supplierPreset/wmsBaseSupplierPreset/component/approvalDialog.vue`
- **功能**:
  - 单个审核：通过/驳回
  - 批量审核：批量通过/批量驳回
  - 驳回时需要输入驳回原因
  - 自动刷新待审核数据
  - 错误处理和数据一致性检查

## 权限控制
- **业务员**: 只能操作自己的申请数据
- **管理员**: 可以操作所有用户的申请数据
- **审核权限**: 只有审核通过的数据才能修改和删除

## 审核流程
1. 业务员提交供应商信息申请
2. 管理员在审核弹窗中查看待审核数据
3. 管理员审核通过后，数据同步到供应商信息表
4. 管理员可以驳回申请，需要填写驳回原因

## 数据权限
- 账号类型为 "999" 或 "888" 的用户（超管和管理员）：可以看到所有用户的申请数据
- 其他账号类型的用户：只能看到自己创建的申请数据

## 错误处理
- 审核不存在的数据会提示"该申请已不存在，已自动刷新列表"
- 审核通过的数据不能删除
- 数据不一致时会自动刷新列表
