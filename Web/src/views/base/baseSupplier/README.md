# 供应商信息管理

## 功能概述
供应商信息管理功能允许管理员查看和管理所有供应商信息，包括已审核通过的和待审核的申请。

## 功能模块

### 1. 供应商信息列表
- **路径**: `/views/base/baseSupplier/index.vue`
- **功能**:
  - 查看所有供应商信息
  - 查询、筛选、排序功能
  - 导入、导出、删除功能
  - 批量操作功能

### 2. 审核功能
- **路径**: `/views/base/baseSupplier/index.vue`
- **功能**:
  - 点击"审核"按钮打开审核弹窗
  - 显示待审核数量
  - 审核通过后同步更新供应商信息表
  - 审核驳回需要填写驳回原因

### 3. 审核弹窗
- **路径**: `/views/base/supplierPreset/wmsBaseSupplierPreset/component/approvalDialog.vue`
- **功能**:
  - 单个审核：通过/驳回
  - 批量审核：批量通过/批量驳回
  - 驳回时需要输入驳回原因
  - 自动刷新待审核数据
  - 错误处理和数据一致性检查

## 权限控制
- **管理员**: 可以查看所有供应商信息，执行审核操作
- **业务员**: 无权限访问此页面

## 审核流程
1. 业务员提交供应商信息申请
2. 管理员在本页面查看所有供应商信息
3. 管理员点击"审核"按钮打开审核弹窗
4. 管理员审核通过后，数据同步到供应商信息表
5. 管理员可以驳回申请，需要填写驳回原因

## 数据权限
- 管理员可以看到所有用户的申请数据
- 只有审核通过的数据才能在本页面显示

## 通知机制
- 页面加载时，如果有待审核数据，显示"待审核通知"
- 定时刷新（每3秒）检查新数据
- 有新数据进来时显示"新申请通知"
- 防抖机制：3秒内只显示一次通知
